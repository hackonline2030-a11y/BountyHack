# Projet BountyHack — contexte Claude

**Instructions opérationnelles (tous agents)** : suivre d’abord **[`Agent.md`](Agent.md)** — il résume les docs obligatoires, la carte `client/` / `server/`, et le comportement attendu (périmètre, secrets, Git, worktrees).

## Nuances utiles ici

- **Langue** : si l’utilisateur écrit en français, répondre en français ; les messages de commit restent en général **anglais** pour les `type(scope): …`, comme décrit dans `CONVENTIONS.md`.
- **Cohérence** : ne pas contredire `CONVENTIONS.md` / `README.md` ; en cas de conflit entre une supposition et ces fichiers, **privilégier les fichiers du dépôt**.
- **Fichier canonique** : éviter de dupliquer longuement les règles dans `Claude.md` — mettre à jour **`Agent.md`** (ou les `.md` référencés) pour que Cursor, Claude et les autres outils restent alignés.
